# 집피티 위젯 타입 전체 비교

## 기존 위젯 (4개)

### 1. price_chart
**용도**: 시간에 따른 가격 변화  
**시각화**: 라인 차트  
**예시 질문**: "래미안 퍼스티지 최근 6개월 시세 변화는?"

### 2. compare_table
**용도**: 여러 단지 비교  
**시각화**: 테이블  
**예시 질문**: "강남구 래미안 단지들 비교해줘"

### 3. complex_card
**용도**: 단지 상세 정보  
**시각화**: 카드  
**예시 질문**: "헬리오시티 정보 알려줘"

### 4. stats_chart
**용도**: 기본 통계 차트  
**시각화**: 기본 차트  
**예시 질문**: (일반적인 통계)

---

## 새 위젯 (3개) - 2025-11-27 추가

### 5. rankings_table ⭐ NEW
**용도**: 순위 기반 데이터 (TOP N)  
**시각화**: 메달 아이콘 + 순위 테이블  
**주요 기능**:
- 메달 이모지 (🥇🥈🥉)
- TrendingUp/Down 아이콘
- metric별 포맷팅 (price, volume, growth, yield)

**예시 질문**:
- "강남구에서 가장 비싼 아파트 TOP 5 알려줘"
- "서울 거래량 많은 단지 TOP 10은?"

### 6. bar_chart ⭐ NEW
**용도**: 지역별/카테고리별 수치 비교  
**시각화**: 수평 막대 그래프  
**주요 기능**:
- 색상 그라데이션 (5단계)
- 애니메이션 전환
- 양수/음수 색상 구분 (%)
- 단위 자동 포맷팅

**예시 질문**:
- "서울 주요 구별 평균 상승률 비교해줘"
- "강남구 vs 서초구 vs 송파구 거래량 비교"

### 7. pie_chart ⭐ NEW
**용도**: 비율/구성 표시  
**시각화**: SVG 도넛 차트 + 범례  
**주요 기능**:
- SVG 도넛 차트
- 6가지 색상 테마
- 퍼센트 자동 계산
- 색상별 범례

**예시 질문**:
- "강남구 아파트 평형별 거래 비율 보여줘"
- "강남권 지역별 거래 구성 비율은?"
